## 🌟 Overview

**JobFinder** is a next-generation job portal designed with a focus on immersive user experience and high-performance interactivity. Built using **React 19** and powered by **Three.js** and **Spline**, it transforms the mundane task of job searching into an engaging journey.

With butter-smooth animations by **GSAP** and **Lenis**, and a robust backend-ready architecture, JobFinder is more than just a job board—it's a career catalyst.

---

## ✨ Key Features

### 👤 For Job Seekers
- **Immersive Onboarding**: A 3D-powered multi-step setup for profiles, skills, and resumes.
- **Smart Job Search**: Filter and find jobs with real-time feedback and advanced categorization.
- **Save & Track**: Bookmark your favorite roles and track application statuses effortlessly.
- **Real-time Communication**: Integrated chat system to connect directly with employers.
- **Personalized Dashboard**: A central hub for notifications, saved jobs, and recommended roles.

### 🏢 For Employers
- **Job Management**: Post, edit, and manage job listings with a dedicated employer dashboard.
- **Applicant Tracking**: View and manage candidate applications efficiently.
- **Company Branding**: Customizable company profile pages to attract top talent.

---

## 🛠️ Tech Stack

### Frontend Core
- **React 19**: Leveraging the latest features for optimal performance and state management.
- **Vite**: Ultra-fast build tool and development server.
- **React Router 7**: Sophisticated routing with protected paths and nested layouts.

### Visuals & 3D
- **Spline & Three.js**: High-fidelity 3D models and interactive scenes.
- **React Three Fiber/Drei**: Bridge between React and Three.js for seamless integration.
- **Lucide React**: Clean and consistent iconography.

### Animations & UX
- **GSAP & Framer Motion**: Industry-leading libraries for complex timeline-based and gesture-based animations.
- **Lenis**: Smooth scroll integration for a premium feel.
- **Barba.js**: Fluid page transitions.
- **CSS Modules**: Scoped and maintainable styling.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/BAJI-761/JOB-FINDER.git
   ```
2. Navigate to the project directory:
   ```bash
   cd JOB-FINDER
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```

### Build
Generate a production-ready bundle:
```bash
npm run build
```

## 🎨 Design Philosophy

The project follows a **"Premium Dark"** aesthetic, utilizing:
- **Glassmorphism**: Subtle translucency for a modern feel.
- **Micro-interactions**: Responsive hover states and transitions.
- **3D Depth**: Interactive Spline scenes in the hero and onboarding sections.
- **Fluid Layouts**: Fully responsive design optimized for all screen sizes.
